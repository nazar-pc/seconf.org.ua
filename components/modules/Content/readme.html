Content module - simple content functionality. May be used by other components or stand-alone

To use module you need few steps:

  • Create content in administration with some key
  • In any theme or component you can get content in such way: 
    use cs\modules\Content\Content;
$content_item	= Content::instance()->get('content key here');
$content	= $content_item['content']; //title field available too
  • Add attribute to container with content: 
    <section data-cs-content="content key here">
	<?=$content?>
</section>
  • If you want to edit content - put mouse cursor over container and Edit button will appear, there is no need to edit it from administration anymore
